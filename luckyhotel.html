About the Project

What Is Lucky Hotel?

Lucky Hotel is a PHP-based hotel reservation system set in the City of Pines — Baguio. It lets guests browse rooms, make bookings, and track their reservation history, all within a sleek, immersive dark interface inspired by a night-time city skyline.

🔐

Authentication

Secure sign-in and sign-up system. Only registered guests can browse rooms and make reservations.

🏨

Room Browsing

Four room types — Single, Premium, Deluxe, and Family — each with photos, descriptions, and nightly rates.

📅

Live Booking

Select dates, guests, room count and type. The total amount auto-calculates before you confirm.

📋

Booking History

View all past reservations in a table. Add new entries or delete existing ones and print the history.
